# Public APIs 4Kr

## 무엇인가

Public APIs 4Kr는 한국 서비스와 공공/민간 데이터 API를 도메인별로 정리한 GitHub 기반 API 카탈로그다.

## 사용 사례

- 한국 서비스 기반 PoC에서 사용할 외부 API 후보를 빠르게 탐색한다.
- agent tool integration에 연결할 수 있는 공개 API 후보를 찾는다.
- RAG나 자동화 워크플로에 필요한 공공데이터, 지도, 금융, 교통, 날씨, 문화, 통계, AI, 쇼핑, 물류, 미디어, 클라우드 API 출처를 찾는다.

## 강점

- 한국 맥락의 API가 한 저장소에 도메인별로 모여 있다.
- 각 항목이 API 설명과 인증 방식을 함께 보여주는 구조다.
- 글로벌 Public API 문서도 별도로 연결한다.

## 한계

- 카탈로그 자체는 각 API의 공식 문서가 아니다.
- 실제 사용 전에는 원 제공자의 최신 문서, 인증 방식, 요금, rate limit, 이용약관을 재확인해야 한다.
- 링크 상태와 API 정책은 시간이 지나며 바뀔 수 있다.

## 관련 워크플로

- PoC 아이디어 단계에서 API 후보군을 빠르게 좁힌다.
- Codex가 특정 한국 서비스 API를 찾을 때 1차 discovery source로 사용한다.
- API를 tool로 연결하기 전, upstream 공식 문서 검증 단계로 이어간다.
