# Obsidian

## 무엇인가

Obsidian은 Markdown note를 탐색하고 편집하기 위한 human-facing interface다.

## 사용 사례

- wiki page 읽기.
- backlink와 graph relationship 탐색.
- Codex가 유지보수한 note 검토.

## 강점

- local Markdown file.
- Obsidian-style link.
- 개인 지식 탐색에 유용함.
