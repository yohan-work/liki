## 2025 publication 대상

- `RegExp.escape`
- Float16 on TypedArrays, DataView, `Math.f16round`
- `Promise.try`
- sync iterator helpers
- JSON modules와 import attributes
- RegExp modifiers
- new Set methods
- duplicate named capture groups

## 2026 publication 대상

- Map·WeakMap upsert
- `JSON.parse` source text access
- iterator sequencing
- Uint8Array base64/hex
- `Math.sumPrecise`
- `Error.isError`
- `Array.fromAsync`

## 2027 publication 대상으로 표시된 항목

- explicit resource management
- `Atomics.pause`
- joint iteration
- Temporal

## 사용 경계

- Finished proposal은 Stage 4와 expected publication year를 확인하는 공식 근거다.
- 이 목록만으로 특정 browser나 Node version의 실제 지원 여부를 판단하지 않는다.
- Expected publication year와 최종 ECMA edition 비준은 구분한다.
- 제안 semantics는 각 proposal specification과 ECMA-262 snapshot을 직접 확인한다.
