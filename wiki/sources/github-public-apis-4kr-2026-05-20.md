## Summary

This repository is a curated catalog of public APIs available for Korean services. The README describes the project as a systematically organized list for developers and says it has been updated with 2026 information. It also links to a separate global public APIs document.

The resource groups APIs by domain, including government and public institutions, maps and location, regional public data, finance and payment, telecom, transportation, weather and environment, healthcare, education, real estate, culture and tourism, statistics and data, AI and machine learning, ecommerce, logistics, media and content, social and communication, energy, science and research, business, cloud services, blockchain, IoT, legal, security, public safety, aviation, agriculture and fisheries, living economy, budget, Naver, and Kakao.
