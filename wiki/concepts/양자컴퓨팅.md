# 양자컴퓨팅

## 한 줄 정의

양자컴퓨팅은 큐비트, 중첩, 얽힘, 측정 같은 양자역학적 성질을 계산 모델에 활용하려는 컴퓨팅 분야다.

## 핵심 문제

고전 컴퓨터는 비트를 0과 1의 상태로 다룬다. 양자컴퓨팅은 큐비트의 상태와 측정 확률, 얽힘을 이용해 고전적 계산 모델과 다른 방식으로 문제를 표현하고 계산하려 한다. 다만 실제 유용성은 문제 유형, 오류 보정, 하드웨어 규모, 알고리즘, 시뮬레이션 비용에 따라 달라진다.

[[Velog Hyunide Bell Inequality Quantum Article]]은 양자컴퓨팅 입문 주제로 Bell's Inequality를 소개한다. 고전적 상관관계에서는 CHSH 합이 `|S| <= 2`를 넘지 못하지만, 얽힘 상태를 사용하면 Tsirelson's bound인 `2√2 ≈ 2.828`에 접근할 수 있다는 설명을 제공한다.

## 좋은 사용 사례

- Bell inequality 같은 기초 실험을 Qiskit으로 재현해 양자 얽힘과 측정의 의미를 학습한다.
- 고전 시뮬레이션으로 충분한 영역과 양자 하드웨어가 필요한 영역을 구분한다.
- 수치해석, 최적화, 시뮬레이션, 암호학, 양자 머신러닝 같은 주제가 실제 문제와 연결되는 조건을 검토한다.

## 실패 패턴

- 양자컴퓨팅을 모든 계산 문제의 일반적 가속기로 오해한다.
- Bell test나 얽힘을 초광속 정보 전달처럼 설명한다.
- 시뮬레이터 결과와 실제 양자 하드웨어 결과의 noise, shot count, backend 차이를 구분하지 않는다.
- Qiskit 예제를 실행했다는 이유만으로 알고리즘의 물리적 의미나 한계를 이해했다고 판단한다.
- "양자 우위" 같은 표현을 구체적 문제, baseline, 하드웨어 조건 없이 사용한다.
